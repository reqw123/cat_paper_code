# 異常門檻設定指南

## 目前的門檻機制

在 **EDA2.py** 中，異常門檻採用 **百分位數自動計算** + **最小值保護** 的雙重機制：

```python
ABNORMAL_PERCENTILE = 0.99  # 自動門檻百分位數
MIN_ABNORMAL_THRES = 0.1    # 最小異常分數門檻
```

### 實際運作方式

```
最終門檻 = max(百分位數門檻, 最小門檻)
```

## 建議設定區間

### 1. 根據檢測目標選擇百分位數

#### 🔴 **嚴格監控（推薦）** - 只關注真正異常行為
```python
ABNORMAL_PERCENTILE = 0.99  # Top 1%
MIN_ABNORMAL_THRES = 0.1    # 保護下限
```
- **結果**: 2幀異常（Frame 181-182）
- **適用**: 正常監控、異常行為檢測
- **特點**: 高精確度、低誤報

#### 🟡 **平衡模式** - 捕捉更多可疑行為
```python
ABNORMAL_PERCENTILE = 0.95  # Top 5%
MIN_ABNORMAL_THRES = 0.1
```
- **結果**: 10幀異常
- **適用**: 行為分析、訓練數據收集
- **特點**: 平衡檢測率和誤報率

#### 🟢 **寬鬆模式** - 廣泛監控
```python
ABNORMAL_PERCENTILE = 0.90  # Top 10%
MIN_ABNORMAL_THRES = 0.1
```
- **結果**: 20幀異常
- **適用**: 初步篩選、數據探索
- **特點**: 高召回率、可能有誤報

### 2. MIN_ABNORMAL_THRES 的作用

這個參數主要用於**保護機制**，防止在數據質量極好時門檻過低：

```python
MIN_ABNORMAL_THRES = 0.1   # 推薦值（針對您的數據）
```

**在您的案例中：**
- 99%分位數 = 15.66（遠大於0.1）
- 所以 MIN_ABNORMAL_THRES 目前**沒有實際作用**
- 除非設定為 > 15.66（例如 20.0），才會影響結果

### 3. 固定門檻模式（不推薦）

如果要使用固定門檻而非百分位數：

```python
ABNORMAL_PERCENTILE = 0.00  # 禁用百分位數
MIN_ABNORMAL_THRES = 10.0   # 使用固定值
```

但**不推薦**，因為：
- 不同視頻的分數範圍差異大
- 無法自動適應數據分布
- 需要頻繁手動調整

## 實際建議

### 對於您的貓咪監控系統

**推薦設定（當前使用）：**
```python
ABNORMAL_PERCENTILE = 0.99  # 99%分位數
MIN_ABNORMAL_THRES = 0.1    # 保護下限
```

**理由：**
1. ✅ **過濾機制已很嚴格**：50%的幀已被質量檢查過濾
2. ✅ **數據質量高**：tracking_quality平均0.94
3. ✅ **減少誤報**：只關注真正的異常行為
4. ✅ **檢測到連續異常**：Frame 181-182連續異常（更可信）

### 如果想要檢測更多異常

可以降低百分位數：
```python
ABNORMAL_PERCENTILE = 0.95  # 改為95%（會得到10幀）
```

### 如果發現遺漏重要異常

1. 檢查 EDA.py 的質量過濾參數：
   ```python
   MIN_VISIBLE_KPT_RATIO = 0.75    # 降低到0.65可保留更多數據
   BORDER_MARGIN = 50              # 降低到30可容忍更靠近邊界
   ```

2. 或降低異常門檻：
   ```python
   ABNORMAL_PERCENTILE = 0.90  # 降低到90%
   ```

## 動態調整示例

如果您想根據異常幀數量動態調整：

```python
# 期望捕捉5-10幀異常
target_abnormal_frames = 7

# 嘗試不同百分位數
for p in [0.90, 0.92, 0.94, 0.95, 0.97, 0.99]:
    thres = frame_scores["score"].quantile(p)
    count = (frame_scores["score"] > thres).sum()
    print(f"p={p:.2f}: {count} frames")
    if count <= target_abnormal_frames:
        ABNORMAL_PERCENTILE = p
        break
```
